## Summary

Get current working directory of the `nimplant` instance.

- Needs Admin: False
- Version: 1
- Author: @NotoriousRebel

## Usage

```
pwd
```

## Detailed Summary

Uses the [getCurrentDir](https://nim-lang.org/docs/os.html#getCurrentDir) proc from Nim's os module.
